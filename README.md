# Task Tracker API

## Описание
Backend API для управления задачами с JWT и Redis.

## Стек
- FastAPI
- PostgreSQL / SQLite
- Redis
- Docker

## Структура проекта
(вставляешь дерево)

## Запуск
docker-compose up --build

## Маршруты
- POST /auth/register
- POST /auth/login
- GET /auth/me
- ...
